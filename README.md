# Pivex Trading Tool 🏆

Copilote IA pour le Pivex Challenge — analyse en temps réel avec signaux de trading.

## Déploiement sur Vercel (5 minutes)

### Étape 1 — Crée un compte GitHub
1. Va sur https://github.com
2. Crée un compte gratuit

### Étape 2 — Crée un dépôt
1. Clique sur "New repository"
2. Nom : `pivex-trading-tool`
3. Clique "Create repository"

### Étape 3 — Upload les fichiers
1. Clique "uploading an existing file"
2. Glisse TOUS les fichiers de ce dossier
3. Clique "Commit changes"

### Étape 4 — Déploie sur Vercel
1. Va sur https://vercel.com
2. Connecte-toi avec GitHub
3. Clique "New Project"
4. Sélectionne `pivex-trading-tool`
5. Clique "Deploy"

### Résultat
Tu obtiens une URL permanente du type :
`https://pivex-trading-tool.vercel.app`

Accessible 24h/24, depuis n'importe quel appareil !
